# Attack-on-Alien
A retro-inspired 2D arcade shooter built with Python and the Pygame library.
Inspired by classics like Galaga, this game throws you into space where waves of alien ships descend toward Earth. Your mission? Destroy them before they destroy you.
Simple mechanics. Fast action. Pure arcade vibes.

## Gameplay Overview
Control a spaceship at the bottom of the screen

Shoot incoming alien enemies

Avoid collisions and enemy attacks

Survive as long as possible

Score points for every alien destroyed

## How to Play (Setup Instructions)
Follow these steps to run the game on your device:

1 - Install Pygame

Make sure Python is installed, then install Pygame using:

pip install pygame

2 - Download:

main_game.py

All asset files (images, sounds, etc.)

3 - Organize Files, Create a folder on your device  and place:

main_game.py

All downloaded asset files inside the same folder.

4 - Run the Game, Open terminal / command prompt inside the folder and run:

python main_game.py

The game window should launch 

## Controls 
Left Arrow - Move Left

Right Arrow - Move Right

SPACE - Shoot
